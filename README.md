## Development server

在启动应用之前运行 `npm i`， 为开发服务器运行`ng serve`， 端口导航到`http://localhost:4300/`.
Run `npm i` before launching APP. Run `ng serve` for a dev server. Port navigate to `http://localhost:4300/`.
